# 激光大冒险 - 迷宫式关卡设计

## 核心概念：关键路径 + 辅助障碍 + 迷宫构型

### 一、激光分类

#### 关键激光（🔴 红色边框标记）
- **作用**：形成从起点到终点的"必经之路"
- **规则**：必须穿越，不可绕过
- **数量**：2-6道，作为难度指标
- **视觉**：红色发光边框，更明显

#### 辅助激光（普通样式）
- **作用**：增加移动困扰和紧张感
- **规则**：可以绕过，但会浪费时间或增加风险
- **效果**：
  - 迫使玩家改变路线
  - 制造时间压力
  - 增加操作复杂度

---

## 二、迷宫构型设计

### 构型A：主路径 + 分支（1-25关）

```
起点
│
╠═══════关键激光1═══════╣ ← 必须穿越
│       │
│   [辅助激光墙]         ← 可绕，但麻烦
│       │
│   ════════关键激光2═══════╣ ← 必须穿越
│       │       │
│    短路径    长路径     ← 分支选择
│   (危险)    (安全)
│       │       │
│       └───────┘
│               │
╠═══════════════关键激光3═══════╣ ← 必须穿越
│                               │
终点
```

**游戏性：**
- 玩家可以选择"短但危险"或"长但安全"的路线
- 辅助激光迫使玩家做选择
- 关键激光确保必须经过N道防线

```
        ═══════关键激光1═══════
       ╱                         ╲
      ╱    [辅助激光障碍]          ╲
     ╱                               ╲
起点 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 终点
     ╲                               ╱
      ╲    ═════关键激光2══════    ╱
       ╲   ╱                 ╲   ╱
        ╲ ╱   [辅助激光]      ╲ ╱
         ╲                     ╱
          ═════关键激光3═══════
```

**游戏性：**
- 玩家必须绕环一周，经过3道关键激光
- 辅助激光布置在环上，增加移动难度
- 可以顺时针或逆时针走

### 构型C：多层迷宫（51-75关）

```
起点
│
╠═══════关键激光1═══════╣ ← 第1层入口
│
│   ┌───────────────┐
│   │  第1层迷宫     │
│   │  [辅助激光]    │
│   │      ↓         │
│   └──────┬──────────┘
│          │
╠══════════关键激光2══════════╣ ← 进入第2层
│
│   ┌───────────────┐
│   │  第2层迷宫     │
│   │  [辅助激光]    │
│   │      ↓         │
│   └──────┬──────────┘
│          │
╠══════════关键激光3══════════╣ ← 出口
│
终点
```

**游戏性：**
- 每层都必须穿越一道关键激光才能进入下一层
- 层内是小型迷宫，有辅助激光增加难度
- 类似于"checkpoint"机制

### 构型D：动态迷宫（76-100关）

```
        ═════关键激光A═════
       ╱    ╲       ╱    ╲
      ╱      ╲     ╱      ╲
起点 ━┓       ═════关键激光B═════  ┏━ 终点
     ┃      ╱                  ╲  ┃
     ┃     ╱   [移动辅助激光]   ╲ ┃
     ┃    ╱                      ╲┃
     ┗━━═关键激光C═══════════════┛
```

**游戏性：**
- 关键激光形成"门框"，必须穿过门框
- 门之间有辅助激光和炸弹
- 部分辅助激光会移动（增加动态难度）

## 三、关键激光生成规则

### 必须保证
1. **连通性**：从起点到终点至少有一条路径
2. **必须经过**：关键激光形成"必经之门"
3. **可穿越性**：关键激光必须有开启时间窗口

### 关键激光布局算法
```
1. 从起点到终点画一条主路径
2. 在主路径上设置N个"门"（关键激光）
3. 每个门都是横向或纵向的完整激光
4. 门之间添加辅助激光和迷宫元素
5. 确保没有其他路径可以绕过所有门
```

---

## 四、辅助激光设计

### 类型1：误导型
- 看起来像关键路径，实际是死胡同
- 玩家走错路会浪费时间

### 类型2：驱赶型
- 迫使玩家向特定方向移动
- 例如：在右侧布置纵向激光，迫使玩家向左走

### 类型3：干扰型
- 在关键激光附近布置，增加穿越难度
- 例如：关键激光是横向的，旁边加纵向辅助激光

---

## 五、难度递进

| 关卡 | 关键激光 | 辅助激光 | 迷宫复杂度 |
|------|---------|---------|-----------|
| 1-10 | 2道 | 2-3道 | 简单直线 |
| 11-25 | 3道 | 3-5道 | 单分支 |
| 26-50 | 3道 | 5-8道 | 环形 |
| 51-75 | 4道 | 6-10道 | 多层 |
| 76-100 | 4-5道 | 8-12道 | 动态混合 |

---

## 六、视觉设计

### 关键激光
- 红色发光边框 + 脉冲效果
- 明显比普通激光更"危险"
- 在UI上标注"必须经过"

### 辅助激光
- 普通颜色（橙/黄/绿/蓝）
- 无特殊边框
- 作为环境障碍
